## Описание проекта

Исходные данные содержат информацию на месяц до оттока и факт оттока на определённый месяц.| *pandas*, *numpy*, *datetime*, *matplotlib*, *seaborn*, *scipy.stats*, *plotly*, sklearn, scipy, matplotlib |
